Data to Visualization with Excel

Document version: 1.0 | Last updated: January 2026

Purpose: This lesson teaches you to create charts in Excel—not just mechanically, but with an understanding of how visualization choices shape interpretation. The same data can tell very different stories depending on how you chart it.

Time: 45-60 minutes

Software: Microsoft Excel (desktop version)

Why This Matters

When AI tools generate charts, they make choices: chart type, axis ranges, colors, labels. These choices aren't neutral—they emphasize some patterns and hide others. To evaluate AI-generated visualizations, you need to understand what choices were made and what alternatives existed.

Building charts yourself gives you that understanding. You'll see firsthand how the same numbers become different stories.

Part 1: The Data

We'll use a simple dataset: quarterly sales for two products over one year.

Quarter Product A Product B
Q1 $120,000 $95,000
Q2 $125,000 $102,000
Q3 $118,000 $108,000
Q4 $130,000 $115,000
Setup:
  1. Open Excel and create a new workbook
  2. Enter the data above starting in cell A1
  3. Format the numbers as currency (select cells, right-click, Format Cells, Currency)

Part 2: Basic Chart Creation

Creating a Column Chart

  1. Select all your data (A1 through C5, including headers)
  2. Go to Insert tab → Charts group → Column Chart
  3. Choose "Clustered Column"

You now have a basic chart comparing both products across quarters.

What Excel did automatically:
  • Set the Y-axis to start at $0
  • Chose axis increments
  • Added a legend
  • Applied default colors

Each of these is a choice that affects how the data is perceived.

Part 3: How Choices Change the Story

Experiment 1: Changing the Y-Axis

Right-click on the Y-axis → Format Axis. Under "Axis Options," find "Bounds."

Try this:
  1. Set Minimum to 0 (default) — What story does this tell?
  2. Set Minimum to 90000 — How does the story change?
  3. Set Minimum to 115000 — Now what do you see?

Key insight: The same data, with different Y-axis starting points, tells completely different stories:

  • Starting at $0: Both products show modest, similar performance
  • Starting at $90,000: Differences between products appear larger
  • Starting at $115,000: Product A's Q3 dip looks like a crisis; Product B appears to barely exist in Q1

None of these charts are "lying"—the numbers are accurate. But each emphasizes different aspects of the data.

Experiment 2: Changing Chart Type

With your chart selected, go to Chart Design tab → Change Chart Type.

Try these chart types with your data:
  1. Line chart — Emphasizes trends over time
  2. Stacked column — Emphasizes total combined sales
  3. 100% stacked column — Shows market share (relative proportions)
  4. Pie chart (select just one quarter) — Shows proportion at one moment

For each: What question does this chart type best answer? What does it hide?

Chart Type Best For Can Hide
Column/Bar Comparing values across categories Trends over time (if not ordered)
Line Showing trends and changes over time Absolute values (focus is on direction)
Stacked Showing parts of a whole over time Individual category performance
Pie Proportions at a single point in time Everything else (trends, comparisons)

Experiment 3: Color and Emphasis

Click on any data series (the bars for one product) → Format Data Series → Fill.

Try this:
  1. Make Product A bright red and Product B gray
  2. Now reverse: Product A gray, Product B bright green

Which product does your eye go to first in each version?

Color isn't neutral. Bright colors draw attention. Red often implies "bad" or "warning." Green implies "good" or "growth." Gray recedes into the background. These associations affect how viewers interpret your chart—even if the data hasn't changed.

Part 4: The "Make It Lie" Challenge

Now you'll create two charts from the same data that support opposite conclusions.

Challenge:

Using only the sales data above, create:

  1. Chart A: Makes Product A look like the clear winner, dominant and successful
  2. Chart B: Makes Product B look like the rising star, the better investment

You may NOT change the numbers. You may only change:

  • Chart type
  • Axis ranges
  • Which data to include/exclude
  • Colors and emphasis
  • Titles and labels

Hints for Chart A (Product A wins)

  • Use total annual sales (Product A: $493,000 vs Product B: $420,000)
  • Start Y-axis at $0 to show the absolute gap
  • Use a simple bar chart comparing totals
  • Title: "Annual Sales Performance"

Hints for Chart B (Product B wins)

  • Focus on growth rate, not absolute numbers
  • Product A grew 8.3% from Q1 to Q4; Product B grew 21%
  • Use a line chart showing the trajectory
  • Start Y-axis at $90,000 to emphasize the closing gap
  • Title: "Quarterly Growth Trends"

The lesson: Neither chart lies about the numbers. Both are "accurate." But they answer different questions and lead viewers to different conclusions. When you evaluate AI-generated charts—or any charts—ask: What question is this chart answering? What choices were made? What alternative stories could this data tell?

Part 5: Connecting to AI Evaluation

When AI tools generate visualizations, they make all these same choices—often without telling you:

  • What chart type to use
  • Where to start the axes
  • What to emphasize with color
  • What data to include or exclude
  • How to title and label

Now that you've made these choices yourself, you can spot them in AI output. Questions to ask:

  1. Why did the AI choose this chart type? Would another type tell a different story?
  2. Where does the Y-axis start? Would starting at zero change the impression?
  3. What's being emphasized? What's being downplayed?
  4. Is anything left out of this visualization?
  5. What question is this chart best at answering? Is that the question I asked?

Excel Skills Reference

Quick reference for chart operations:

Task How
Create chart Select data → Insert → Charts → Choose type
Change chart type Select chart → Chart Design → Change Chart Type
Edit axis Right-click axis → Format Axis
Change colors Click data series → Format → Fill
Add title Chart Design → Add Chart Element → Chart Title
Add data labels Chart Design → Add Chart Element → Data Labels
Move chart Click and drag, or Chart Design → Move Chart
