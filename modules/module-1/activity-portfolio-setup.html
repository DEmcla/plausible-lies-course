Portfolio Setup

Purpose: Create the structure for your Learning Portfolio document. This document will grow throughout the course as you add artifacts and reflections.

Time: 30-45 minutes

Software: Microsoft Word (desktop version required)

What You're Building

Your portfolio is a single Word document that demonstrates your growth in evaluating AI and technology. By setting up the structure now, you'll:

  • Have a place to organize your work throughout the course
  • Learn professional Word skills (styles, TOC, formatting)
  • Avoid scrambling to learn these skills under deadline pressure later

Setup Instructions

Step 1: Create a New Document

  1. Open Microsoft Word (desktop application)
  2. Create a new blank document
  3. Save it as LastName_Portfolio.docx

Step 2: Create a Cover Page

  1. Go to Insert → Cover Page
  2. Choose a simple, professional design
  3. Fill in:
    • Title: "Learning Portfolio"
    • Subtitle: "CSCI 247: Confident Uncertainty"
    • Your name
    • Date (you can update this with each submission)

Step 3: Insert a Page Break

  1. Click at the end of your cover page
  2. Go to Insert → Page Break (or press Ctrl+Enter / Cmd+Enter)
  3. This ensures your Table of Contents starts on a new page

Step 4: Add Section Headings

Type these headings, each on its own line. After typing each one, select it and apply Heading 1 style (Home tab → Styles):

  1. Content Knowledge
  2. Problem Solving
  3. Communication
  4. Graphs & Visualization
  5. Scientific Practices
  6. Source Evaluation
  7. Synthesis Reflection

Why Heading 1? Using heading styles (not just bold text) allows Word to automatically generate and update your Table of Contents. This is a key professional skill.

Step 5: Add Placeholder Text

Under each heading, add placeholder text so you remember what goes there:

  • Under each outcome heading: "[Artifact and reflection to be added]"
  • Under Synthesis Reflection: "[Final reflection to be added at end of course]"

Step 6: Insert Table of Contents

  1. Click right after your cover page (on the blank page before your first heading)
  2. Go to References → Table of Contents
  3. Choose Automatic Table 1 or Automatic Table 2
  4. Your headings should appear in the TOC

If your headings don't appear: Make sure you applied the Heading 1 style (not just made the text bold). Select each heading and verify "Heading 1" is highlighted in the Styles panel.

Step 7: Test the TOC

  1. Add a new heading somewhere in your document (e.g., "Test Heading") and apply Heading 1
  2. Go back to your Table of Contents
  3. Right-click on it and select Update Field → Update entire table
  4. Your new heading should appear
  5. Delete the test heading and update the TOC again

Checklist: What Your Document Should Have

  • ☐ Cover page with title, course name, your name, date
  • ☐ Table of Contents on its own page
  • ☐ Seven Heading 1 sections (six outcomes + synthesis)
  • ☐ Placeholder text under each heading
  • ☐ TOC correctly lists all your headings
  • ☐ TOC updates when you add/change headings
